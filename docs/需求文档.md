# 高考志愿性格测试 — 需求文档

## 1. 项目背景

每年高考出分后，大量考生面临志愿填报的困惑：不知道自己喜欢什么、适合什么、未来想做什么。市面上的性格测试要么收费、要么过于学术化、要么不接地气。本项目旨在提供一个免费、好用、贴合中国高考场景的自测工具。

## 2. 目标用户

- 高考后准备填报志愿的高中生
- 在校高中生（提前了解自己）
- 家长和教育工作者（辅助了解孩子）

## 3. 核心需求

### 3.1 自我认知测试
- 30道选择题，覆盖6大维度（实践操作、研究探索、艺术创造、社会助人、经营影响、常规事务）
- 每题5级评分（非常不符合 ~ 非常符合）
- 测试过程友好、轻松，适合高中生

### 3.2 测试结果分析
- 六大维度评分雷达图（直观可视化）
- 综合解读：你的性格画像（文字描述）
- 推荐职业类型（Top 3）
- 对应专业门类（与国内大学专业分类对应）
- 工作类型建议（白领/蓝领/技术员/工程师/科研人员/创意工作者等）
- 个性化填报建议

### 3.3 用户体验
- 无需注册、无需登录
- 完全离线运行，无隐私泄露风险
- 响应式设计，手机和电脑均可使用
- 界面美观、年轻化，符合高中生审美

## 4. 技术方案

- 纯前端单页应用（SPA）
- HTML + CSS + JavaScript
- Canvas API 绘制雷达图
- 无任何外部依赖
- 模块化 JS 结构

## 5. 输出物

- `src/index.html` — 主页面
- `src/css/style.css` — 样式文件
- `src/js/questions.js` — 题库定义
- `src/js/results.js` — 结果分析引擎
- `src/js/app.js` — 应用核心逻辑
- `docs/需求文档.md` — 本文件
- `README.md` — 项目概述
