# project-001-高考志愿性格测试

## 项目目标
帮助高考考生在填报志愿前更好地了解自己——通过专业的兴趣、性格、能力测试，找到适合自己的职业方向、大学专业门类和工作类型。

## 项目状态
进行中

## 技术栈
- 纯前端 HTML + CSS + JavaScript（无需后端、无需安装、无外部依赖）
- Canvas 雷达图可视化
- 完全本地运行，保护隐私

## 目录说明
- docs/：需求文档
- src/：源代码（index.html + css/ + js/）
- assets/：资源文件（图片等）
- outputs/：输出产物

## 使用方法
直接用浏览器打开 `src/index.html` 即可开始测试。建议使用 Chrome 或 Edge 浏览器。
- **本地使用**：直接用浏览器打开 `src/index.html` 即可开始测试
- **在线访问**：已配置 GitHub Actions 自动部署，推送到 GitHub 后自动上线

## 在线部署
本项目已配置 GitHub Pages 自动部署，推送到 GitHub 后自动发布到：
`https://你的用户名.github.io/gaokao-career-test/`
（实际链接取决于你的 GitHub 用户名和仓库名称）

## 备注
本测试基于霍兰德职业兴趣理论（RIASEC）改编，结合中国高考志愿填报的实际场景进行本土化设计。
